# Instrucciones de Instalación y Ejecución del Proyecto

## 1. Descargar el Proyecto

1. Dirígete al botón **Code** y selecciona **Download ZIP** para descargar el proyecto.

   ![Botón Code](https://github.com/user-attachments/assets/6deef183-a5a2-4ec3-8a28-3fad29ea4d9e)

## 2. Importar el Proyecto en NetBeans

1. Una vez descargado el archivo ZIP, descomprímelo.
2. Abre NetBeans y selecciona **File > Open Project**.
3. Navega hasta la carpeta descomprimida y selecciona el proyecto para abrirlo.

   ![NetBeans Open Project](https://github.com/user-attachments/assets/b483cc4b-41ac-4ee9-b8ed-b7e862ddc3bd)

## 3. Ejecutar el Proyecto

1. Con el proyecto abierto en NetBeans, presiona la tecla **F6** para compilar y ejecutar el proyecto.
